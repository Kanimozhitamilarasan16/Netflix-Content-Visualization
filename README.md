# Netflix Content Visualization
INTERN ID:CITS5189
## Objective
The objective of this project is to visualize and analyze the Netflix Titles dataset using Python.

## Tools Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Dataset
Netflix Titles Dataset (netflix_titles.csv)

## Features
- Data Cleaning
- Movies vs TV Shows Visualization
- Top 10 Countries
- Content Released by Year
- Top 10 Genres
- Ratings Distribution
- Movie Duration Distribution
- Correlation Heatmap

## Conclusion
This project provides insights into Netflix's content library through data visualization techniques, helping understand content distribution, trends, and characteristics.
